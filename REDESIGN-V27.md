# YR Finanças v27 — Redesign real

Esta versão mantém o projeto verdadeiro, com a lógica já existente de:

- Supabase/login/sincronização;
- dados locais como apoio;
- cartões/formas de pagamento;
- compras;
- parcelas automáticas;
- conferência de fatura;
- pessoas/devedores;
- recebimentos;
- estabelecimentos;
- categorias;
- configurações e backup.

O que foi feito nesta versão:

- Novo visual global inspirado na prévia aprovada.
- Sidebar lateral fixa no desktop.
- Layout responsivo para tablet e celular.
- Bottom navigation no celular.
- Cards, tabelas, filtros, formulários e painéis redesenhados.
- Service worker e referências atualizadas para v27.

Recomendação: testar localmente antes de subir para o GitHub Pages, principalmente login/Supabase e cache do PWA.
